### Broken Flows

**1. Teams Channel Alert Delivery** (affects TEAM-03)
- Path: `dispatch()` → `sendChannelAlert({ channelId: "19:xxx@thread.tacv2" })` → `teamConversationReferences["19:xxx@thread.tacv2"]` → `undefined` → silent no-op
- Root cause: `storeConversationReference` in `teams-bot-handler.ts` keys by `ref.conversation.tenantId` (AAD GUID), but `sendChannelAlert` looks up by channel thread ID
- Fix: Key channel-scoped refs by `ref.conversation.id` instead of `tenantId`

**2. Onboarding Wizard OAuth Connect** (affects ONBD-01)
- Path: Click "Connect" → `window.open("/api/oauth/google-workspace/authorize")` → 404
- Root cause: Route `/api/oauth/${slug}/authorize` doesn't exist; only `/api/oauth/[provider]/callback/route.ts` exists
- Fix: Use `trpc.integration.getOAuthUrlGeneric({ provider })` and redirect popup to returned URL
