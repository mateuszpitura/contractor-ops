# External Enablement Register

> **Principle:** external/manual dependencies never block development. Build the code, gate the dependent path behind a **default-off `module.*` flag (kill switch)**, make tests that need the external artifact **conditionally skip** (auto-flip GREEN when it lands), and record the enablement here. Nothing ships enabled without the artifact **and** an adviser/legal sign-off where noted. See [`MEMORY`](./MEMORY.md) â†’ *External deps never block dev*.

**Dev bypass:** the boot signoff gate for gated flag namespaces is bypassed locally with `FLAG_SIGNOFF_BYPASS=local` (LOCAL-ONLY â€” never in prod). Flags default disabled; flip to APPROVED post-deploy.
