    "use client" component. Props: `slaDeadline: string | null`, `status: string`, `slaHours?: number`.

    Compute SLA status client-side from the server-provided slaDeadline and status:
    - If no deadline or status !== "PENDING", return null
    - Calculate remainingMs = new Date(slaDeadline).getTime() - Date.now()
    - If remainingMs <= 0: overdue. Hours = Math.ceil(Math.abs(remainingMs) / 3600000). Label = `OVERDUE ${hours}h`
    - If > 0: hours = Math.ceil(remainingMs / 3600000). Compute percentage using slaHours if provided, otherwise estimate.
    - Color thresholds per D-08: >50% = green, 25-50% = yellow, <25% = red

    Badge styling per UI-SPEC SLA countdown badge colors:
    - green: `bg-success/10 text-success` (use CSS variable --success or tailwind green-500)
    - yellow: `bg-warning/10 text-warning` (use amber-500)
    - red: `bg-destructive/10 text-destructive`
    - overdue: `bg-destructive/[0.15] text-destructive border border-destructive/30`

    Minimum width 72px (per UI-SPEC spacing exceptions) to prevent layout shift.

    Use `useEffect` with `setInterval(60000)` to update countdown every minute.
