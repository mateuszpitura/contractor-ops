  <action>
Create `packages/api/src/services/doc-link-service.ts` with these exported functions. Use loosely typed PrismaClient parameter (precedent: Phase 16, 18) for parallel execution compatibility.

**attachDocLink(prisma, input: { organizationId, integrationConnectionId, workflowTaskRunId, externalId, externalUrl, externalType, metadata }):**
- Create ExternalLink record with:
  - `entityType: "WORKFLOW_TASK_RUN"` (per D-03 — workflow steps only)
  - `entityId: input.workflowTaskRunId`
  - `externalType: input.externalType` ("NOTION_PAGE" or "CONFLUENCE_PAGE")
  - `externalId: input.externalId`
  - `externalUrl: input.externalUrl`
  - `metadataJson: input.metadata` (title, icon, lastEditedTime per D-04)
  - `organizationId`, `integrationConnectionId`
- Return the created ExternalLink

**detachDocLink(prisma, input: { organizationId, externalLinkId }):**
- Delete ExternalLink where `id = externalLinkId AND organizationId = organizationId` (tenant isolation)
- Return void

**getDocLinks(prisma, input: { organizationId, workflowTaskRunId }):**
- Find all ExternalLinks where `entityType: "WORKFLOW_TASK_RUN"`, `entityId: workflowTaskRunId`, `organizationId`, AND `externalType IN ["NOTION_PAGE", "CONFLUENCE_PAGE"]`
- Return array of ExternalLink records

**searchDocs(input: { organizationId, query, provider, prisma }):**
- If provider is "notion" or "all": find NOTION IntegrationConnection for org (status CONNECTED), get decrypted credentials via `getDecryptedCredentials` from integrations package, call `NotionAdapter.searchPages(accessToken, query)`, map results to `DocSearchResult[]` with `provider: "notion"`
- If provider is "confluence" or "all": find CONFLUENCE IntegrationConnection, get credentials, get cloudId from `configJson.cloudId`, call `ConfluenceAdapter.searchPages(accessToken, cloudId, query)`, map to `DocSearchResult[]` with `provider: "confluence"`
- Merge results (Notion first, then Confluence) and return. Max 10 results per provider.
- If no connections exist for a provider, return empty array for that provider (no error)

**refreshDocMetadata(prisma, externalLinkId, organizationId):**
- Find ExternalLink, check if metadataJson.lastEditedTime is older than 24 hours
- If stale: fetch fresh metadata from provider API (Notion get page / Confluence get content)
- Update metadataJson with fresh data
- Return updated ExternalLink
- If not stale: return existing ExternalLink unchanged

Import NotionAdapter and ConfluenceAdapter directly (not via registry) for searchPages calls. Import `getDecryptedCredentials` from `@contractor-ops/integrations` for token retrieval.
