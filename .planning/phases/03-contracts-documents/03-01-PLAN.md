Create `packages/api/src/routers/contract.ts` following the exact same patterns as `contractor.ts`:

**Contract status transition map** (at top of file):
```typescript
const CONTRACT_TRANSITIONS: Record<string, string[]> = {
  DRAFT: ["ACTIVE", "TERMINATED"],
  PENDING_SIGNATURE: ["ACTIVE", "TERMINATED"],
  ACTIVE: ["EXPIRING", "TERMINATED", "SUPERSEDED"],
  EXPIRING: ["ACTIVE", "EXPIRED", "TERMINATED", "SUPERSEDED"],
  EXPIRED: ["TERMINATED", "SUPERSEDED"],
  TERMINATED: [],
  SUPERSEDED: [],
  ARCHIVED: [],
};
```

**Router procedures** (all use `tenantProcedure` + `requirePermission`):

1. **create** — mutation, input: contractCreateSchema. Permission: `requirePermission({ contracts: ["create"] })`. Creates contract with all fields. Returns plain(contract) with contractor included.

2. **getById** — query, input: z.object({ id: z.string() }). Permission: `requirePermission({ contracts: ["read"] })`. Includes contractor (select: id, legalName, companyName, status), amendments (orderBy effectiveDate desc), and document count via `_count` on DocumentLink where entityType=CONTRACT and entityId=contract.id. Returns plain(contract).

3. **update** — mutation, input: z.object({ id: z.string(), data: contractUpdateSchema }). Permission: `requirePermission({ contracts: ["update"] })`. Updates contract fields. Returns plain(updated).

4. **list** — query, input: contractListSchema. Permission: `requirePermission({ contracts: ["read"] })`. Implements:
   - Full-text search via `searchVector @@ to_tsquery('simple', searchTerms)` using `prisma.$queryRawUnsafe` or Prisma where clause with raw SQL (follow contractor.ts FTS pattern)
   - Sorting by sortBy/sortOrder
   - Filtering by status[], type[], billingModel[], ownerUserId[], endDateFrom/To, complianceRiskLevel[], contractorId
   - Pagination with skip/take
   - Include contractor (select: id, legalName, companyName), internalOwner (select: id, name)
   - Return `{ items: plain(contracts), totalCount, page, pageSize }`

5. **transitionStatus** — mutation, input: contractStatusTransitionSchema. Permission: `requirePermission({ contracts: ["update"] })`. Validates transition against CONTRACT_TRANSITIONS map. If TERMINATED, sets terminatedAt = now(). Returns plain(updated).

6. **createAmendment** — mutation, input: amendmentCreateSchema. Permission: `requirePermission({ contracts: ["update"] })`. Creates ContractAmendment linked to contract. Auto-generates amendmentNumber as `AME-{count+1}`. Returns plain(amendment).

7. **listAmendments** — query, input: z.object({ contractId: z.string() }). Permission: `requirePermission({ contracts: ["read"] })`. Returns amendments ordered by effectiveDate desc.

8. **updateExpiryReminders** — mutation, input: contractExpiryReminderSchema. Permission: `requirePermission({ contracts: ["update"] })`. Updates contract.metadataJson to include `{ reminderDaysBefore: [30, 60, 90] }`. Uses Prisma JSON update.

9. **delete** — mutation, input: z.object({ id: z.string() }). Permission: `requirePermission({ contracts: ["delete"] })`. Only allows deletion if status is DRAFT. Soft-deletes (sets deletedAt). Returns { success: true }.

10. **bulkTransition** — mutation, input: z.object({ ids: z.array(z.string()), targetStatus: z.string() }). Permission: `requirePermission({ contracts: ["update"] })`. Validates all transitions. Uses `prisma.$transaction()`. Returns { updated: count, failed: failedIds[] }.
