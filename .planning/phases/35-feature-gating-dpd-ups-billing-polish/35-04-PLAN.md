    **CreditProgressBar** (`credit-progress-bar.tsx`):

    "use client" component. Props: `used: number`, `total: number`.

    Color logic per UI-SPEC:
    ```typescript
    function getBarColor(used: number, total: number): string {
      if (total === 0) return "var(--destructive)";
      const remaining = total - used;
      const pct = remaining / total;
      if (pct > 0.5) return "var(--success)";     // >50% remaining: green
      if (pct >= 0.2) return "var(--warning)";     // 20-50%: yellow
      return "var(--destructive)";                  // <20%: red
    }
    ```

    Render: shadcn `<Progress>` component with custom bar color applied via CSS variable or style prop. Show numeric label: `"{remaining} of {total} credits remaining"` using `useTranslations("Billing.credits")`. Use `aria-valuenow={total - used}`, `aria-valuemin={0}`, `aria-valuemax={total}`.

    When used === total (exhausted): show `t("exhausted")` = "No credits remaining -- purchase more to continue OCR processing".
