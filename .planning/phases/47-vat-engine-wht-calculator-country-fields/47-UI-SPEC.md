## Phase-Specific UI Components

### 1. VAT Rate Selector (Invoice Form)

**Location:** Invoice creation/edit form, replaces hardcoded dropdown
**Component:** `Select` (shadcn) with grouped options by category
**Behavior:**
- Options loaded from tRPC `tax.getRates` filtered by org country
- Groups: "Standard Rates", "Reduced Rates", "Exempt/Special"
- Each option shows: rate code + percentage + description (e.g., "23% - Standard rate")
- Default pre-selected based on `isDefault` flag from TaxRate table
- Disabled state when no rates available for org country

**Layout:**
```
[VAT Rate ▼                    ]
  Standard Rates
    23% — Standard rate        ●
  Reduced Rates
    8% — Reduced rate
    5% — Reduced rate
  Exempt
    0% — Zero rate
    ZW — Tax exempt
    NP — Not applicable
```

### 2. Reverse Charge Banner (Invoice Detail)

**Location:** Invoice detail view, below the line items section
**Component:** `Alert` (shadcn) with `info` variant
**Trigger:** Displayed when invoice has `isReverseCharge: true`
**Layout:**
```
┌─────────────────────────────────────────────┐
│ ℹ  Reverse charge applied                   │
│    Cross-border B2B transaction.             │
│    VAT to be accounted by the buyer.         │
│    [Override ▾]                               │
└─────────────────────────────────────────────┘
```
- Info icon (`Info` from lucide) tinted `--info`
- "Override" dropdown: "Remove reverse charge" / "Keep reverse charge"
- Background: `bg-info/5`, border: `border-info/20`

### 3. Country Compliance Section (Contractor Profile)

**Location:** Within existing contractor profile tabs, in the "Details" or "Compliance" tab
**Component:** Collapsible `Card` section
**Behavior:**
- Section visible only when org `countryCode` matches a supported country (AE, SA)
- Fields rendered conditionally per country
- Validation errors inline per field

**UAE Fields Layout:**

**Saudi Fields Layout:**

─────────────────────────────────

Commercial Registration   [________________]

- Section heading: `text-base font-semibold` (16px)
- Field labels: `text-sm font-medium` (14px)
- Field spacing: `sm` (8px) between label and input, `md` (16px) between field groups
- Missing required fields show `--warning` border tint

## Interaction Patterns

### VAT Rate Selection
- Invoice form loads rates on mount via tRPC query
- Selected rate updates `vatAmountMinor` calculation in real-time
- If org country changes (settings), cached rates are invalidated

### Reverse Charge Toggle
- Auto-detected on invoice creation based on seller/buyer country
- User can override via dropdown on the reverse charge banner
- Override is logged in audit trail

### WHT Certificate Generation
- Triggered per-item or bulk from payment run detail
- Async generation (loading state on button)
- Success: toast notification + certificate appears in documents list
- Failure: inline error in the WHT summary card

### Country Fields Save
- Standard form submission with Zod validation
- Optimistic update with rollback on failure
- Fields only editable by users with contractor edit permissions

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| VAT rate selector keyboard nav | shadcn `Select` has built-in keyboard support |
| Reverse charge banner screen reader | `role="alert"` with descriptive `aria-label` |
| WHT certificate modal focus trap | shadcn `Dialog` handles focus trap |
| Country fields form labels | Explicit `htmlFor` on all labels |
| Monetary values screen reader | `aria-label` with full currency name (e.g., "Saudi Riyal 10,000") |
| Color contrast | All semantic colors pass WCAG AA on their respective backgrounds |
| Status indicators | Text labels accompany all status icons (not icon-only) |
