<!-- From Plan 01 - Backend -->
tRPC router (packages/api/src/routers/ocr.ts):
- ocr.trigger: mutation({ documentId, storageKey, invoiceId? }) => { extractionId }
- ocr.getResult: query({ extractionId }) => OcrExtraction (with resultJson containing OcrExtractionResult)
- ocr.getByDocument: query({ documentId }) => OcrExtraction | null
- ocr.retrigger: mutation({ extractionId }) => { extractionId }
- ocr.portalTrigger: same as trigger but portal-scoped
- ocr.portalGetResult: same as getResult but portal-scoped
- ocr.portalGetByDocument: same as getByDocument but portal-scoped

OcrExtractionResult (from packages/integrations/src/types/ocr.ts):
```typescript
interface OcrExtractionResult {
  status: "EXTRACTED" | "PARTIAL" | "FAILED";
  fields: Record<string, OcrExtractionField>;
  lineItems: OcrLineItem[];
  processingTimeMs: number;
  pageCount: number;
  overallConfidence: number;
}
```

- Right panel: Card with white background containing the form
