## Unsatisfied Requirements Detail

### PAY-06 — LPCDA Late Payment Interest (Phase 63)

**Blocker:** `late-payment-interest.ts` uses `requireFeatureFlag('PAY_LATE_INTEREST_ENABLED')` on all 6 procedures. The key `'PAY_LATE_INTEREST_ENABLED'` is **not** in the `FlagKey` union (correct key: `'payments.late-interest-enabled'`). TypeScript compile error on all 6 procedures.

**Fix required:**
```
packages/api/src/routers/late-payment-interest.ts
Lines 52, 182, 285, 345, 390, 567:
  requireFeatureFlag('PAY_LATE_INTEREST_ENABLED')
→ requireFeatureFlag('payments.late-interest-enabled')
```

**Secondary issue (WR-02 warning, not blocking):** `daysOverdue` computed from `dueDateMs` instead of `overdueStartMs` — label mismatch, legally contestable in LPCDA claim letters.

### PAY-07 — Skonto Early Payment Discount (Phase 63)

**Blocker:** `skonto.ts` router line 287 references `invoice.amountMinor`. The `Invoice` Prisma model has `amountToPayMinor` (not `amountMinor`). `amountMinor` exists only on `InvoicePayment` model. All Skonto monetary calculations produce `NaN` at runtime.

**Fix required:**
```
packages/api/src/routers/skonto.ts line 287:
  invoiceTotalMinor: invoice.amountMinor
→ invoiceTotalMinor: invoice.amountToPayMinor
```
