## Integration Check Results

### Cross-Phase Wiring: 33/35 requirement paths wired

**Critical Gap: OAuth callback missing adapter registration**
- **File:** `/apps/web/src/app/api/oauth/[provider]/callback/route.ts`
- **Issue:** Route calls `getAdapter(provider)` but never calls `registerAllAdapters()`. Registry is empty at callback time — every OAuth connect flow silently fails.
- **Affected requirements:** INTG-01, JIRA-01, CAL-01, SIGN-01, DOCS-01/DOCS-02 (initial connection)
- **Fix:** Add `registerAllAdapters()` call at module top

**High: react-pdf CSS build failure**
- **File:** `/apps/web/src/components/ocr/pdf-viewer.tsx` lines 5-6
- **Issue:** CSS imports from `react-pdf/dist/esm/Page/` not handled by Next.js — blocks admin OCR review panel build
- **Affected requirements:** OCR-03 (admin side-by-side review)
- **Fix:** Add `react-pdf` to `transpilePackages` in `next.config.ts` or dynamic import with `ssr: false`
