## Critical Gap: OCR Adapter Registry

**Root cause:** Phase 20-01 (commit `90fa950`) rewrote `register-all.ts` to add 4 new adapters (Notion, Confluence, Google Calendar, Outlook Calendar) and accidentally dropped the `ClaudeOcrAdapter` registration call that Phase 16-01 had added.

**Fix:** Re-add two lines to `packages/integrations/src/adapters/register-all.ts`:
```ts
import { ClaudeOcrAdapter } from "./claude-ocr-adapter.js";
registerAdapter(new ClaudeOcrAdapter() as unknown as IntegrationProviderAdapter);
```

**Impact:** OCR-01 (auto-extract fields from uploaded PDF) broken at runtime. OCR UI and service code intact — only the adapter registry wiring is missing.
