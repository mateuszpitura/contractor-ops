| collection variables -> a real key | The collection ships `{{apiKey}}` as a variable, never an embedded key |

## STRIDE Threat Register
| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-101-04-02 | Information Disclosure | A real API key committed in a collection | mitigate | Auth is a `{{apiKey}}` variable, never a literal; a test asserts no `co_live_`/`co_test_` literal |
| T-101-04-SC | Tampering | A conversion library day-zero dep | accept | Prefer the hand-rolled mapper (no new dep); any converter obeys the 7-day floor |
