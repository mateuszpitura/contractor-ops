parallel with 100-06). Turns the `api-key-leak-alarm` RED net green.

| leaked key → attacker | Multi-IP usage is the detection signal for a key exfiltrated from a customer |

| T-100-07-02 | Spoofing | Forged XFF inflates/suppresses the IP count | mitigate | Read only the P99 proxy-trusted `sourceIp`, not a client header |
| T-100-07-03 | Information Disclosure | Alarm leaks the plaintext key | mitigate | Alert carries the key prefix + count only, never the secret |
