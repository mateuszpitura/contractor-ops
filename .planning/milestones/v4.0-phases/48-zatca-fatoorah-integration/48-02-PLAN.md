Create `packages/einvoice/src/profiles/zatca/signer.ts`:

Export `ZatcaXAdESSigner` class implementing `Signable`:

**sign(xml: string, certificate: CertificateInfo): Promise<string>**

1. Validate privateKey exists — throw `Error("ZATCA signing requires privateKey in CertificateInfo")` if missing.

2. Parse X.509 certificate from Base64 using `new crypto.X509Certificate(Buffer.from(certificate.certificate, 'base64'))`. Extract serial number, issuer DN, and compute SHA-256 digest of the DER certificate bytes.

3. Build XAdES QualifyingProperties XML:
   ```xml
   <xades:QualifyingProperties xmlns:xades="http://uri.etsi.org/01903/v1.3.2#" Target="#signature">
     <xades:SignedProperties Id="xadesSignedProperties">
       <xades:SignedSignatureProperties>
         <xades:SigningTime>{ISO 8601 UTC}</xades:SigningTime>
         <xades:SigningCertificate>
           <xades:Cert>
             <xades:CertDigest>
               <ds:DigestMethod Algorithm="http://www.w3.org/2001/04/xmlenc#sha256"/>
               <ds:DigestValue>{base64 SHA-256 of DER cert}</ds:DigestValue>
             </xades:CertDigest>
             <xades:IssuerSerial>
               <ds:X509IssuerName>{issuer DN}</ds:X509IssuerName>
               <ds:X509SerialNumber>{serial}</ds:X509SerialNumber>
             </xades:IssuerSerial>
           </xades:Cert>
         </xades:SigningCertificate>
       </xades:SignedSignatureProperties>
     </xades:SignedProperties>
   </xades:QualifyingProperties>
   ```

4. Configure `xml-crypto` `SignedXml`:
   - signingKey = PEM private key
   - Canonicalization: `http://www.w3.org/2001/10/xml-exc-c14n#`
   - Signature algorithm: `http://www.w3.org/2001/04/xmldsig-more#ecdsa-sha256`
   - Reference URI="" (enveloped): transforms [enveloped-signature, exc-c14n], digest SHA-256
   - Reference URI="#xadesSignedProperties": type `http://uri.etsi.org/01903#SignedProperties`, digest SHA-256
   - Custom KeyInfoProvider: `<ds:X509Data><ds:X509Certificate>{base64 cert}</ds:X509Certificate></ds:X509Data>`
   - ds:Object containing XAdES QualifyingProperties

5. Insert `<ds:Signature>` into `<ext:UBLExtensions><ext:ExtensionContent>` in the XML.

6. Return signed XML.

**verify(xml: string): Promise<SignatureVerificationResult>**
- Locate `<ds:Signature>`, extract cert from KeyInfo, validate with SignedXml.checkSignature()
- Extract signing time from xades:SigningTime, signer name from cert subject CN
- Return { valid, signerName, signedAt, errors }

**Security:** Never log privateKey. Generic error messages. Buffer-based key handling.
