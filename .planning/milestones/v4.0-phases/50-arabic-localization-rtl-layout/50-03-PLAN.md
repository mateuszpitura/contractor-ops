  **Per D-05 (Systematic `<bdi>` wrapping):**

  1. **Create `apps/web/src/components/ui/bdi.tsx`:**
     ```tsx
     import { type HTMLAttributes, forwardRef } from "react";
     import { cn } from "@/lib/utils";

     export interface BdiProps extends HTMLAttributes<HTMLElement> {
       children: React.ReactNode;
     }

     /**
      * Bidirectional text isolation component.
      * Wraps content in a <bdi> element to isolate text direction from surrounding context.
      * Use for all user-generated content displayed in RTL-capable layouts:
      * contractor names, invoice numbers, free-text fields, entity references.
      */
     export const Bdi = forwardRef<HTMLElement, BdiProps>(
       ({ children, className, ...props }, ref) => (
         <bdi ref={ref} className={cn(className)} {...props}>
           {children}
         </bdi>
       )
     );
     Bdi.displayName = "Bdi";
     ```

  2. **Wrap user-generated content in data display components:**

     For each file, import `{ Bdi } from "@/components/ui/bdi"` and wrap these content types in `<Bdi>`:
     - **Entity names:** contractor names, vendor names, company names, party names
     - **Invoice numbers:** invoice number references, PO numbers
     - **Free-text fields:** descriptions, notes, comments displayed as read-only
     - **Reference codes:** contract titles, payment references

     **What NOT to wrap:**
     - Static translated UI strings (these come from next-intl, direction handled by document dir)
     - Numeric values (dates, amounts — these use Intl formatting)
     - Status labels (translated strings)
     - Icon-only content

     **Invoice columns.tsx example:**
     - Wrap the invoice number cell value: `<Bdi>{invoice.number}</Bdi>`
     - Wrap the vendor/supplier name cell: `<Bdi>{invoice.vendorName}</Bdi>`

     **Contractor side panel:**
     - Wrap contractor name display: `<Bdi>{contractor.name}</Bdi>`
     - Wrap company name: `<Bdi>{contractor.companyName}</Bdi>`

     **Contract components:**
     - Wrap contract title: `<Bdi>{contract.title}</Bdi>`
     - Wrap party names: `<Bdi>{contract.partyName}</Bdi>`
