## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input → tRPC | Form inputs (TRN, API key) validated client-side and server-side |
| tRPC response → UI | Query responses rendered safely (no dangerouslySetInnerHTML) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-49-15 | Information Disclosure | API key field | mitigate | Input type="password" by default; show/hide toggle explicit user action; never persist in component state after wizard closes |
| T-49-16 | Spoofing | Disconnect action | mitigate | AlertDialog confirmation required before disconnect; mutation uses protectedProcedure auth |
| T-49-17 | Information Disclosure | QR display | accept | QR contains public invoice data (seller name, TRN, amounts) — already visible on the invoice itself |
</threat_model>
