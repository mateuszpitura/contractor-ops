**use-template-form.ts** -- Custom hook wrapping React Hook Form. Uses useForm with zodResolver against a local schema mirroring templateCreateSchema (same local-schema pattern as contractor/contract wizards). Tracks dirty state via formState.isDirty. Adds beforeunload listener when dirty. Provides: form, isDirty, tasks (useFieldArray), addTask (appends defaults: title="", taskType="MANUAL", sortOrder=tasks.length, required=false, assigneeMode="ROLE_BASED"), removeTask (by index, undo toast via sonner 5s), reorderTasks (updates sortOrder after drag).

**condition-builder.tsx** -- Props: value (ConditionGroup|null), onChange. Null state: "No conditions" text + "Add condition" Button. With conditions: vertical rows, each with field Select (8 options: contractor.type/status/billingModel/team/complianceRiskLevel, contract.type/status/currency), operator Select (equals/notEquals/contains/startsWith), value Input or Select (enum fields get Select with values, strings get Input). X button per row. AND/OR toggle between rows (toggles entire group combinator). "Add condition" below. All labels via useTranslations("Workflows").

**task-card.tsx** -- Collapsible Card. Props: task, index, onRemove, allTasks, form. Collapsed (56px): drag handle area (GripVertical 24px), title (or "Untitled task" muted), type Badge, assignee label, due offset, required Badge, condition badge ("Only if: {summary}" 240px max-width ellipsis). Click to expand. Expanded: title Input, taskType Select (10 options), description Textarea, assigneeMode Select (5 options), conditional role Select (when ROLE_BASED, 8 UserRole options), conditional user Select (when FIXED_USER, from tRPC user.list), dueOffsetDays/Hours number Inputs, required Switch, dependency Select (only tasks with lower sortOrder + "No dependency"), ConditionBuilder. "Done editing" Button, "Remove task" destructive link. All via useTranslations("Workflows").

**sortable-task-list.tsx** -- DndContext + SortableContext from @dnd-kit. verticalListSortingStrategy. Each TaskCard in sortable wrapper. Drag listeners on GripVertical handle ONLY (not whole card). onDragEnd: arrayMove + onReorder callback. "Add task" secondary Button at bottom. Empty: "Add your first task to build this workflow."

**template-form.tsx** -- Props: templateId? (undefined=new). If templateId, fetch via trpc.workflow.getTemplate, populate form via reset(). Header: large Input for name, Select for type (5 options), Textarea for description, status Badge. Actions top-right: "Save template" primary (dot when isDirty), "Activate" (when DRAFT), "Archive" (when ACTIVE, AlertDialog), "Duplicate", "Delete" (DRAFT only, AlertDialog). Below: SortableTaskList. Save calls createTemplate (new) or updateTemplate (edit). Success: toast + redirect. Error: toast.
