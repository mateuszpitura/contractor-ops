    Create packages/api/src/services/invoice-matching.ts with two exported functions:

    a) computeDuplicateCheckHash(invoiceNumber: string, sellerTaxId: string, totalGrosze: number): string
       - Uses Node crypto: createHash("sha256").update(`${invoiceNumber.trim().toLowerCase()}|${sellerTaxId.trim()}|${totalGrosze}`).digest("hex")

    b) runAutoMatch(prisma, organizationId, invoice: { sellerTaxId, totalGrosze, currency, duplicateCheckHash }, deviationThresholdPercent = 10): Promise<MatchResult>
       - Step 1: Match contractor by NIP: prisma.contractor.findFirst where nip === sellerTaxId AND organizationId AND deletedAt is null. If no sellerTaxId or no match, return { matchStatus: "UNMATCHED", contractorId: null, contractId: null, score: 0, flags: [] }.
       - Step 2: Find active contracts: prisma.contract.findMany where contractorId AND organizationId AND status in ["ACTIVE", "EXPIRING"].
       - Step 3: Score calculation: exact NIP match = 50 points. If active contract found = +30 points. For each contract, compute expected amount from rateValueGrosze (stored in contract customFieldsJson or direct field). Pick contract with closest expected amount. If amount within threshold = +20 points. Score 80-100 = "MATCHED", 50-79 = "PARTIAL", 0-49 = "UNMATCHED".
       - Step 4: Deviation: amountDeltaGrosze = invoice.totalGrosze - expectedAmountGrosze. amountDeltaPercent = (delta / expected) * 100. If abs(percent) > threshold, matchStatus = "DISCREPANCY".
       - Step 5: Flags in flagsJson array: "NO_ACTIVE_CONTRACT" if no contracts found, "EXPIRED_CONTRACT" if best contract status is EXPIRED, "CURRENCY_MISMATCH" if contract currency !== invoice currency.
       - Step 6: Duplicate check: prisma.invoice.findFirst where duplicateCheckHash AND organizationId AND id !== currentInvoiceId. If found, add "DUPLICATE_SUSPECTED" to flags.
       - Return { matchStatus, contractorId, contractId, score, expectedAmountGrosze, amountDeltaGrosze, amountDeltaPercent, flags, duplicateInvoiceId }.
  </action>

    1. create: tenantProcedure + requirePermission({ invoice: ["create"] }). Input: invoiceCreateSchema. Creates Invoice record in RECEIVED status with source MANUAL_UPLOAD. Computes duplicateCheckHash via computeDuplicateCheckHash if sellerTaxId and invoiceNumber provided. Creates InvoiceFile records linking each documentId with role SOURCE_ORIGINAL. Also creates DocumentLink records with entityType INVOICE. Returns plain(invoice).

    2. getById: tenantProcedure + requirePermission({ invoice: ["read"] }). Input: z.object({ id: z.string() }). Includes contractor (select id, legalName, nip), contract (select id, title, type, status, rateValueGrosze, currency), files with document, matchResults. Returns plain(invoice).

    3. update: tenantProcedure + requirePermission({ invoice: ["update"] }). Input: z.object({ id: z.string(), data: invoiceUpdateSchema }). Only allows update when status is RECEIVED. Recomputes duplicateCheckHash on relevant field changes. Returns plain(invoice).

    4. list: tenantProcedure + requirePermission({ invoice: ["read"] }). Input: invoiceListSchema. Server-side pagination with where clause from filters (status in, matchStatus in, source in, contractorId). Search via invoiceNumber contains OR contractor legalName contains (case-insensitive). OrderBy from sortBy/sortOrder. Include contractor (select id, legalName). Returns { items: plain(invoices), totalCount, page, pageSize }.

    5. statusCounts: tenantProcedure + requirePermission({ invoice: ["read"] }). No input. Uses prisma.invoice.groupBy on status and matchStatus for organizationId. Returns Record<string, number> with combined status/matchStatus counts.

    6. submitForMatching: tenantProcedure + requirePermission({ invoice: ["update"] }). Input: z.object({ id: z.string() }). Validates invoice is in RECEIVED status. Reads org settingsJson for invoiceDeviationThresholdPercent (default 10). Calls runAutoMatch. Creates InvoiceMatchResult record. Updates invoice with contractorId, contractId, matchStatus, status (UNDER_REVIEW), flagsJson. Uses prisma.$transaction for atomicity. Returns plain(updatedInvoice).

    7. manualMatch: tenantProcedure + requirePermission({ invoice: ["update"] }). Input: invoiceManualMatchSchema. Validates contractor and contract belong to org. Creates InvoiceMatchResult with matchedBy MANUAL, status MANUALLY_CONFIRMED. Updates invoice with contractorId, contractId, matchStatus MANUALLY_CONFIRMED. Returns plain(invoice).

    8. voidInvoice: tenantProcedure + requirePermission({ invoice: ["delete"] }). Input: z.object({ id: z.string() }). Sets status to VOID. Returns plain(invoice).

    9. dismissDuplicate: tenantProcedure + requirePermission({ invoice: ["update"] }). Input: z.object({ id: z.string() }). Removes DUPLICATE_SUSPECTED from flagsJson array. Returns plain(invoice).

    10. searchContractors: tenantProcedure + requirePermission({ invoice: ["read"] }). Input: z.object({ query: z.string().min(1) }). Searches contractors by legalName or nip containing query (case-insensitive). Limit 10. Returns plain(contractors) with id, legalName, nip, status.
