<objective>
Build the reports page with sidebar navigation, date range filter, 5 report types (spend by contractor, spend by team, expiring contracts, overdue invoices, compliance gaps), summary charts with drill-down, detail tables, and CSV export.

Purpose: Give users filterable, exportable reports on spend, compliance, and contract status.
Output: Complete /reports page with all 5 report types, charts, tables, drill-down, and CSV export.

RPT-02 scope note: The "spend by team" report groups via Contractor.primaryTeamId -> Team. While the schema has Contract.projectId and Contract.costCenterId fields, "spend by project" and "spend by cost center" groupings are deferred since no UI for managing projects/cost centers exists yet. The team grouping satisfies the primary use case; project and cost center groupings can be added as follow-up report types once those management UIs are built.
</objective>
