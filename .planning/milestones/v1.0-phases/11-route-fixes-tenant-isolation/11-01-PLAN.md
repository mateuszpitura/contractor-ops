Each of the 4 list pages needs to read a `?action=` URL searchParam and auto-open the corresponding dialog/wizard when the param is present. All pages already use nuqs and are wrapped in Suspense. Use `useQueryState` from nuqs to read the `action` param, then use a `useEffect` to trigger the dialog open state when the param matches.

**Pattern to apply on each page (inside the Content component):**

```typescript
import { parseAsString, useQueryState } from "nuqs";

// Inside the Content component, after existing state declarations:
const [action, setAction] = useQueryState("action", parseAsString);

// After existing state, add useEffect to auto-trigger:
useEffect(() => {
  if (action === "ACTION_VALUE") {
    setDialogOpen(true);
    // Clear the action param so refreshing doesn't re-trigger
    void setAction(null);
  }
}, [action, setAction]);
```
