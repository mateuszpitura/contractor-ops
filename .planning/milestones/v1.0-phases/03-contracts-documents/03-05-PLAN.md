**drop-zone.tsx** — Reusable drag-and-drop component using react-dropzone:
- Dashed border container (2px dashed `--border`), min-height 160px
- Centered UploadCloud icon (32px), "Drag files here or" text, "Browse files" button

- Props: `onFilesAccepted: (files: File[]) => void`, `onFileRejected?`, `disabled?`, `entityType?`, `entityId?`
- ACCEPTED_TYPES map: PDF, DOCX, XLSX, PNG, JPG with MIME types
- MAX_FILE_SIZE: 25 MB
- Multiple files accepted
- All text via `useTranslations("Documents")`

**document-card.tsx** — Single document display card:
- File type icon (48x48 area, use FileText/FileSpreadsheet/Image icons from Lucide based on MIME)
- Filename, version badge ("Version N"), upload date, uploader name
- Scan status inline badge: Scanning (Loader2 spinning + muted), Clean (ShieldCheck green), Infected (ShieldAlert red), Failed (ShieldQuestion amber)
- Actions: Download button (calls `trpc.document.getDownloadUrl`, opens URL in new tab), Preview button (PDF only, opens PdfPreview dialog), "Upload new version" link
- For infected documents: Download button disabled with tooltip "Threat detected"

**document-list.tsx** — List of document cards:
- Props: `entityType`, `entityId` — fetches via `trpc.document.list`
- Renders DocumentCard for each document
- Empty state when no documents

**pdf-preview.tsx** — Dialog for inline PDF preview:
- Dialog max-width 960px, max-height 80vh
- Browser-native `<object>` tag (not react-pdf) with presigned URL as data source
- Download button in dialog header
- Close button top-right
- Fallback message if browser cannot render PDF

**upload-progress.tsx** — Individual file upload progress row:
- File icon (by type), filename, file size, progress bar (0-100%), cancel button (X)
- On complete: replace progress bar with scan status badge
- Uses XMLHttpRequest for progress tracking during direct R2 PUT

**version-history.tsx** — Expandable version history section:
- "View version history" link that expands to show all prior versions
- Each version: version number, upload date, uploader. Prior versions downloadable
- Fetches via `trpc.document.getVersionHistory`
