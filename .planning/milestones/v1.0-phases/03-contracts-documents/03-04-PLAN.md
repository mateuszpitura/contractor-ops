**wizard-dialog.tsx** — Dialog (640px max-width) with:
- 3-step progress indicator at top: "Contract details" (step 1), "Financial terms" (step 2), "Documents" (step 3)
- Completed steps show checkmark icon, current step highlighted with `--primary`, future steps grayed
- Single react-hook-form with zodResolver using local Zod schema that mirrors contractCreateSchema from validators (same pattern as contractor wizard — local schema to avoid cross-package dependency)
- Step navigation: "Next: Financial terms" / "Next: Documents" / "Create contract" CTAs
- "Back" button on steps 2-3
- Per-step validation: step does not advance until fields valid
- Props: `open`, `onOpenChange`, `contractorId?: string` (for pre-fill when opened from contractor profile)
- When contractorId provided: contractor field auto-selected and read-only, fetch contractor's billing profile via `trpc.contractor.getById` and pre-fill financial terms (rateValueGrosze, currency, billingModel)
- Pre-filled fields show hint text: "Pre-filled from contractor billing profile" (translated)
- Dismiss logic: pristine form shows "Close", modified form shows "Discard changes" with confirmation dialog
- On submit: call `trpc.contract.create.mutate()`, on success close dialog, show sonner toast "Contract created", invalidate contract list query

**step-details.tsx** — Step 1 form fields:
- Contractor picker (Combobox/Select with search, fetching from `trpc.contractor.list`). Required. Read-only if contractorId prop provided.
- Contract title (Input, required)
- Contract type (Select with all ContractType enum values, translated)
- Start date (Calendar date picker, required)
- End date (Calendar date picker, optional)
- Notice period days (Input number, optional)
- Auto-renewal (Checkbox)

**step-financial.tsx** — Step 2 form fields:
- Rate (Input number, required, display as zloty but store as grosze — divide by 100 for display, multiply by 100 for storage)
- Currency (Select: PLN, EUR, USD)
- Billing model (Select with BillingModel enum values, translated)
- Rate type (Select with RateType enum values, translated)
- Payment terms days (Input number, required)
- Invoice cycle (Select with InvoiceCycle enum values, translated)
- Show "Pre-filled from contractor billing profile" hint below each pre-filled field (only when pre-filled from contractor)

**step-documents.tsx** — Step 3:
- Basic drag-and-drop file input using react-dropzone directly (reusable DropZone component is created in Plan 03-05)
- Shows uploaded files with filename, size, progress
- "Skip, add documents later" link below drop zone
- Documents uploaded immediately on selection (not deferred to form submit) — call `trpc.document.requestUpload` + direct PUT to presigned URL + `trpc.document.confirmUpload`
- Each file shows upload progress bar, then scan status badge once confirmed
- Link uploaded documents to the newly created contract via `trpc.document.linkToEntity` after contract creation
