<objective>
Implement the GitHub `Deprovisionable` adapter — the one with a fundamentally different authorization model. Using `@octokit/rest`, the adapter:
1. `suspendAccount` → `orgs.removeMember` removes the user from the org, all teams, and org-level access (CONTEXT.md D-06).
2. `revokeAllSessions` → on SAML SSO orgs, enumerates and revokes the user's per-PAT credential authorizations (CONTEXT.md D-04); on non-SAML orgs, surfaces a warning WITHOUT failing the run.
3. Flags outside-collaborator repos (which survive org removal — Pitfall 7 "outside-collab back-door") as a MANUAL item with direct links (CONTEXT.md D-05).
4. `verifyDeprovisioned` → membership 404 / state !== active → LIKELY_GONE (D-14).
5. `describeImpact` → repo/team/outside-collab/invitation/PAT/owner metrics.

Output: 1 adapter class + GitHub tests flipped GREEN.
</objective>

<action>
Create `packages/integrations/src/adapters/github-adapter.ts`. Read `slack-adapter.ts` (org-level raw/token flow + Deprovisionable method shape from Phase 77) and `google-workspace-adapter.ts` (DeprovisionResult/hash/classify body) first.

Class shape:
```ts
import { Octokit } from '@octokit/rest';
export class GitHubAdapter extends BaseAdapter implements Deprovisionable {
  readonly slug = 'github';
  readonly displayName = 'GitHub';
  readonly supportsOAuth = true; // GitHub App installation (D-09)
  readonly supportsWebhooks = false;
  // private #octokit(auth): new Octokit({ auth })  — installation/classic token from decrypted credentials
  // the org login is stored on the connection (credentials/metadata)
  async suspendAccount(externalUserId: string): Promise<DeprovisionResult> { ... }   // externalUserId = GitHub username/login
  async revokeAllSessions(externalUserId: string): Promise<DeprovisionResult> { ... }
  async verifyDeprovisioned(externalUserId: string): Promise<boolean> { ... }
  async describeImpact(externalUserId: string): Promise<ImpactPreview> { ... }
}
```

**suspendAccount(username):** `await octokit.rest.orgs.removeMember({ org, username })`. 204/2xx → SUCCEEDED. 404 → LIKELY_GONE (already not a member). Classify other errors via `classifyError('GITHUB', err.status, body, headers)`. Hashes via canonicalize (descriptor `{ op:'removeMember', org, username }`; never the token).

**revokeAllSessions(username):**
1. Enumerate `const auths = await octokit.paginate('GET /orgs/{org}/credential-authorizations', { org })` inside a try/catch.
   - On 403/404 (non-SAML org / endpoint unavailable) → return SUCCEEDED with a descriptive detail `'Per-PAT revocation unavailable (org not on SAML SSO)'` and `authorizedPatCount` semantics null. Do NOT FAIL.
2. For each `a` with `a.login === username`: `await octokit.request('DELETE /orgs/{org}/credential-authorizations/{credential_id}', { org, credential_id: a.credential_id })` (cap concurrency with `p-limit`, e.g. 5). Aggregate failures via classifier; a transient failure on one PAT marks the step FAILED for retry, a permanent one surfaces the warning.
3. SUCCEEDED when all reachable PATs revoked (or none / non-SAML). Record count revoked + `token_last_eight` list (masked) in audit.

**verifyDeprovisioned(username):** `await octokit.rest.orgs.checkMembershipForUser({ org, username })`; a 404 → true (not a member). If it resolves 204 (still a member) → false. Optionally `getMembershipForUser` state !== 'active' → true. Never throw (errors → false).

**describeImpact(username):** GITHUB ImpactPreview member:
- org membership / `getMembershipForUser` → isOrgOwner (`role === 'admin'`), accountStatus.
- `octokit.paginate` org repos the user can access / team repos → repositoryCount (best-effort).
- `octokit.paginate('GET /orgs/{org}/outside_collaborators', { org })` → list; if the username is an outside collaborator, set outsideCollaboratorRepoCount to the repos they collaborate on (enumerate per-repo collaborator membership best-effort) and include direct repo links in a manual-task descriptor.
- teams → teamMembershipCount; pending invitations → `octokit.paginate('GET /orgs/{org}/invitations')` filtered → pendingOrgInvitations.
- credential-authorizations count (SAML only) → authorizedPatCount (null if non-SAML).
- cacheKey `co:idp:preview:GITHUB:{username}`, fetchedAt ISO. Total preview failure → throw typed error for Phase 77 D-03 proceed-without-preview.

Audit via `getIdpAuditLogger()`. No `console.*`. Verify the exact Octokit method names/`paginate` route strings against the installed `@octokit/rest` types if they differ.
