Export `async function processKsefSync(params: { organizationId: string; connectionId: string; }): Promise<{ invoicesCreated: number; duplicatesFound: number; errors: string[] }>`:

Flow (per RESEARCH.md sync orchestrator pattern):
1. **Create IntegrationSyncLog** with status "STARTED", direction "INBOUND", syncType "ksef_invoice_fetch".
2. **Decrypt KSeF credentials**: `const connection = await prisma.integrationConnection.findUniqueOrThrow({ where: { id: params.connectionId } })`. Verify `connection.organizationId === params.organizationId`. Decrypt via `decryptCredentials(connection.credentialsRef, "ksef")`. Parse config from `connection.configJson` via `ksefConnectionConfigSchema.parse(connection.configJson)`.
3. **Get org NIP** (per D-03): `const org = await prisma.organization.findUniqueOrThrow({ where: { id: params.organizationId } })`. Extract NIP from `(org.settingsJson as Record<string, unknown>)?.taxId as string`. If NIP is falsy, throw Error "Organization NIP not configured. Set it in Organization Settings."
4. **Authenticate with KSeF**: `const client = new KsefApiClient(config.environment ?? "prod")`. If `config.authMethod === "token"`: `await client.authenticate(credentials.accessToken, nip)`. If certificate: `await client.authenticateWithCertificate(credentials.extra?.certificateBase64 as string, credentials.extra?.certificatePassword as string | undefined, nip)`.
5. **Determine date range**: `dateFrom = connection.lastSuccessAt ? connection.lastSuccessAt.toISOString().split("T")[0] : new Date(Date.now() - 90 * 24 * 3600 * 1000).toISOString().split("T")[0]` (default last 90 days for first sync, per pitfall 4). `dateTo = new Date().toISOString().split("T")[0]`.
6. **Query invoices**: `const result = await client.queryInvoices(nip, dateFrom, dateTo)`.
7. **Process each invoice metadata**:
   - Download XML: `const xml = await client.downloadInvoiceXml(metadata.ksefReferenceNumber)`
   - Parse: `const parsed = parseFa3Xml(xml, metadata.ksefReferenceNumber)`
   - Map to Invoice fields: `const fields = mapKsefToInvoiceFields(parsed)`
   - Compute duplicate hash: `const hash = computeDuplicateCheckHash(fields.invoiceNumber, fields.sellerTaxId!, fields.totalGrosze)`
   - **Check cross-source duplicate** (per D-11): `const dup = await checkCrossSourceDuplicate(prisma, params.organizationId, fields.invoiceNumber, fields.sellerTaxId!)`
   - Create Invoice: `const invoice = await prisma.invoice.create({ data: { ...fields, organizationId: params.organizationId, duplicateCheckHash: hash, status: "RECEIVED", matchStatus: "UNMATCHED", approvalStatus: "NOT_STARTED", paymentStatus: "NOT_READY", lines: { create: fields.lines.map(l => ({ ...l, organizationId: params.organizationId })) } } })`
   - If duplicate found: `await linkDuplicateInvoices(prisma, invoice.id, dup.existingInvoiceId!)` (per D-12)
   - **Run auto-match** (per D-08): `await runAutoMatch(prisma, params.organizationId, { id: invoice.id, sellerTaxId: fields.sellerTaxId, totalGrosze: fields.totalGrosze, currency: fields.currency, duplicateCheckHash: hash })`
   - Track counts: increment invoicesCreated, duplicatesFound, collect errors per invoice.
8. **Terminate KSeF session**: `await client.terminateSession()` in finally block.
9. **Update IntegrationConnection**: `lastSyncAt: new Date()`, `lastSuccessAt: new Date()` (on success), `lastErrorAt` + `lastErrorMessage` (on error). Update `status: errors.length > 0 && invoicesCreated === 0 ? "ERROR" : "CONNECTED"`.
10. **Update IntegrationSyncLog**: `status: "SUCCESS"` or `"FAILED"`, `completedAt: new Date()`, `responsePayloadJson: { invoicesCreated, duplicatesFound, errors }`.
11. **Dispatch batch notification** (per D-10): If `invoicesCreated > 0`, call `dispatch({ organizationId: params.organizationId, type: "KSEF_SYNC_COMPLETE", title: "KSeF Sync Complete", message: \`${invoicesCreated} new invoices fetched from KSeF\`, link: "/invoices?source=KSEF" })`.
12. Return `{ invoicesCreated, duplicatesFound, errors }`.

Wrap the entire body in try/catch. On catch: update sync log to FAILED, update connection to ERROR, re-throw.

Use `@upstash/redis` distributed lock (pattern from Phase 12 token refresh) to prevent concurrent syncs for the same org: `const lockKey = \`ksef-sync:\${params.organizationId}\``, acquire with 5-minute TTL, release in finally.

Follow the exact pattern from `apps/web/src/app/api/ocr/_process/route.ts`:

```typescript
import { NextResponse } from "next/server";
import type { NextRequest } from "next/server";
import { verifySignatureAppRouter } from "@upstash/qstash/nextjs";
import { registerAllAdapters } from "@contractor-ops/integrations/adapters/register-all";
import { processKsefSync } from "@contractor-ops/api/services/ksef-sync-orchestrator";

registerAllAdapters();

async function handler(request: NextRequest) {
  const body = await request.json();
  const { organizationId, connectionId } = body as {
    organizationId: string;
    connectionId: string;
  };

  if (!organizationId || !connectionId) {
    return NextResponse.json(
      { error: "Missing organizationId or connectionId" },
      { status: 400 },
    );
  }

  try {
    const result = await processKsefSync({ organizationId, connectionId });
    return NextResponse.json({ processed: true, ...result });
  } catch (error) {
    console.error(
      `[ksef/_sync] Failed to sync KSeF for org ${organizationId}:`,
      error,
    );
    return NextResponse.json(
      { error: "KSeF sync failed" },
      { status: 500 },
    );
  }
}

export const POST = verifySignatureAppRouter(handler);
```
