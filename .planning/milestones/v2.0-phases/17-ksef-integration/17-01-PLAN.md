**1. Create `packages/integrations/src/services/ksef-api-client.ts`**:

```typescript
import { createPublicKey, publicEncrypt, constants, createDecipheriv } from "node:crypto";
```

Export `interface KsefSession { jwt: string; referenceNumber: string; encryptionKey: Buffer; }`.

Export `interface KsefInvoiceMetadata { ksefReferenceNumber: string; invoiceNumber: string; subjectNip: string; invoiceDate: string; }`.

Export `interface KsefQueryResult { invoiceMetadataList: KsefInvoiceMetadata[]; hasMore: boolean; pageToken?: string; }`.

Export `class KsefApiClient`:

- Constructor takes `environment: "test" | "prod"` (default "prod"). Set `this.baseUrl` to `"https://ksef.mf.gov.pl/api/v2"` for prod, `"https://ksef-test.mf.gov.pl/api/v2"` for test.

- Private `session: KsefSession | null = null`.

- `async authenticate(token: string, nip: string): Promise<KsefSession>`:
  1. `GET ${baseUrl}/auth/public-key` -- extract publicKey PEM from response
  2. `POST ${baseUrl}/auth/challenge` with body `{ contextIdentifier: { type: "nip", value: nip } }` -- get `{ challenge, timestampMs }`
  3. RSA-OAEP encrypt: `const rsaKey = createPublicKey(publicKey)`, `const plaintext = Buffer.from(token + "|" + timestampMs)`, `const encrypted = publicEncrypt({ key: rsaKey, padding: constants.RSA_PKCS1_OAEP_PADDING, oaepHash: "sha256" }, plaintext)`
  4. `POST ${baseUrl}/auth/token/redeem` with body `{ challenge, encryptedToken: encrypted.toString("base64") }` -- get `{ jwt, referenceNumber }`
  5. Poll `GET ${baseUrl}/auth/${referenceNumber}` with 1s interval (max 30 attempts) until status field indicates ready
  6. Store session, return `{ jwt, referenceNumber, encryptionKey: Buffer.alloc(32) }` (encryption key extraction from session response if available)
  - All fetch calls include `Content-Type: application/json` header. Authenticated calls include `Authorization: Bearer ${session.jwt}`.
  - On HTTP error, throw descriptive Error with status code and KSeF error message.

- `async authenticateWithCertificate(certificateBase64: string, password: string | undefined, nip: string): Promise<KsefSession>`:
  - Decode Base64 to buffer, extract private key from .p12 using password
  - Sign XAdES challenge XML using private key
  - Submit to `POST ${baseUrl}/auth/certificate/redeem`
  - Follow same polling pattern as token auth
  - Mark as `// TODO: Full XAdES implementation -- token auth is primary path per D-01`

- `async queryInvoices(nip: string, dateFrom: string, dateTo: string): Promise<KsefQueryResult>`:
  - Requires active session (throw if not authenticated)
  - `POST ${baseUrl}/invoices/query/metadata` with body `{ queryCriteria: { subjectNip: nip, invoicingDateFrom: dateFrom, invoicingDateTo: dateTo, subjectType: "subject2" } }` -- get `{ queryId }`
  - Poll `GET ${baseUrl}/invoices/query/${queryId}/status` until done (max 60 attempts, 2s interval)
  - Extract metadata list from completed query response
  - Return parsed `KsefQueryResult`

- `async downloadInvoiceXml(ksefReferenceNumber: string): Promise<string>`:
  - Requires active session
  - `GET ${baseUrl}/invoices/ksef/${ksefReferenceNumber}` with Authorization header
  - Handle AES-256 decryption if response is encrypted (check content-type)
  - Return raw XML string

- `async verifyCredentials(token: string, nip: string): Promise<boolean>`:
  - Try authenticate, return true on success, false on error. Used for D-04 credential verification.

- `async terminateSession(): Promise<void>`:
  - If session exists, `POST ${baseUrl}/auth/session/terminate` with Authorization header
  - Clear session

- Private helper `async fetchWithRetry(url: string, options: RequestInit, retries = 2): Promise<Response>`:
  - Handles HTTP 429 with Retry-After header (exponential backoff)
  - Handles HTTP 5xx with retry
  - Throws on 4xx (except 429)
