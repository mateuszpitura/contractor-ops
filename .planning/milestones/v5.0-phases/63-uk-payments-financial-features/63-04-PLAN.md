**Create `packages/api/src/routers/bacs.ts` (per D-27):**

Import the existing `tenantProcedure` (or equivalent org-scoped procedure) from the router infrastructure. Import `requirePermission` middleware.

**Procedures:**

1. `previewExport` — `tenantProcedure.input(z.object({ paymentRunId: z.string() })).query(...)`:
   - Load PaymentRun with items, each item's invoice + contractor + billingProfile.
   - Check PAY_BACS_ENABLED feature flag; throw FORBIDDEN if off.
   - Check org BACS submitter fields exist; throw FAILED_PRECONDITION with message 'BACS submitter not configured' if missing.
   - Decrypt UK sort codes + account numbers for each item via `decryptBankAccount()`.
   - Decrypt org submitter fields.
   - Call `generateBacsStandard18(items, orgBank, runRef, new Date())`.
   - Return `{ fileText: result.fileBuffer.toString('ascii'), transliterationWarnings: result.transliterationWarnings, modulusWarnings: result.modulusWarnings }`.

2. `generateExport` — `tenantProcedure.input(z.object({ paymentRunId: z.string() })).mutation(...)`:
   - Same load + decrypt as previewExport.
   - Call `generateBacsStandard18(...)`.
   - Compute SHA-256 hash of fileBuffer.
   - Upload to R2: key = `payment-exports/{organizationId}/{paymentRunId}/BACS-{runNumber}-{sha256Prefix16}.txt`.
   - Create `PaymentExport` row: format=`BACS_STD18`, r2Key, sha256, downloadCount=1.
   - Generate signed download URL (TTL 300s) via `signExistingDownload`.
   - Return `{ downloadUrl, filename: '{orgName}-BACS-{runNumber}-{isoDate}.txt', sha256 }`.

3. `validateSortCode` — `tenantProcedure.input(z.object({ sortCode: sortCodeSchema, accountNumber: accountNumberSchema })).query(...)`:
   - Import `modulusCheck` from `@contractor-ops/validators` + `VOCALINK_MODULUS_TABLE_V840`.
   - Run modulus check.
   - Return `{ status: 'VALID' | 'WARN' | 'INVALID', warnings: string[] }`.
   - 'INVALID' only for regex-level failures (should not happen if Zod schema passes). 'WARN' for exception-category sort codes.

4. `saveSubmitterConfig` — `tenantProcedure.use(requirePermission('org:settings:write')).input(z.object({ serviceUserNumber: serviceUserNumberSchema, submitterSortCode: sortCodeSchema, submitterAccountNumber: accountNumberSchema, submitterName: bacsSubmitterNameSchema })).mutation(...)`:
   - Encrypt each field via `encryptBankAccount()`.
   - Compute masked versions: SUN last 2 (`XXXX{last2}`), sort code last 2 (`XX-XX-{last2}`), account last 4 (`XXXX{last4}`).
   - Update Organization with encrypted + masked fields + plain submitterName.
   - Log audit entry (field names changed, never values).
   - Return `{ saved: true, masks: { sun, sortCode, accountNumber } }`.

**Wire into appRouter in `packages/api/src/routers/index.ts`:**
- Add `bacs: bacsRouter` to the router merge.
