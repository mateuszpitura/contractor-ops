**Create `packages/api/src/routers/late-payment-interest.ts` (per D-27):**

1. `getForInvoice` — `tenantProcedure.input(z.object({ invoiceId: z.string() })).query(...)`:
   - Load invoice + contractor (with isBusinessCustomer, countryCode) + all InvoicePayments + all InvoiceInterestWaivers + InvoiceInterestCompensation.
   - Load BoEBaseRateHistory (all entries, global — no org scope).
   - Check PAY_LATE_INTEREST_ENABLED; if off, return `{ featureFlagOff: true }`.
   - If scope gates fail (non-GB, B2C, non-GBP), return `{ applicable: false, reason }`.
   - Ensure compensation tier exists (idempotent upsert): if invoice is overdue and no InvoiceInterestCompensation row exists, create one with `tierMinor = getCompensationTier(invoice.totalMinor)`, `invoiceTotalAtOverdueMinor = invoice.totalMinor`, `firstOverdueDate = invoiceDueDate + 1 day`.
   - Call `calculateLateInterest(...)` with all loaded data.
   - Return the full `LateInterestResult` + `compensationId` + `waiverStatus` + `claimStatus`.

2. `getForOrg` — `tenantProcedure.input(z.object({ status: z.enum(['ALL', 'ACCRUING', 'CLAIMED', 'WAIVED']).optional(), cursor: z.string().optional() })).query(...)`:
   - Paginated list of overdue GB B2B invoices with computed interest.
   - Used by dashboard tile and invoices list.

3. `waive` — `tenantProcedure.input(z.object({ invoiceId: z.string(), waiveType: z.enum(['STATUTORY_INTEREST', 'COMPENSATION', 'BOTH']), reason: z.string().min(10) })).mutation(...)`:
   - Create InvoiceInterestWaiver row. Set `waivedByUserId` from session. Return `{ waiverId }`.

4. `revokeWaiver` — `tenantProcedure.input(z.object({ waiverId: z.string(), revokeReason: z.string().min(10) })).mutation(...)`:
   - Set `revokedAt = now()`, `revokedByUserId` from session, `revokeReason`. Return `{ revoked: true }`.

5. `claim` — `tenantProcedure.input(z.object({ invoiceId: z.string(), issueAsSecondaryInvoice: z.boolean().default(false) })).mutation(...)`:
   - Compute current interest via `calculateLateInterest(...)`.
   - Generate claim PDF via React-PDF render of `LatePaymentClaimTemplate` (see below).
   - Upload PDF to R2: key = `late-interest-claims/{orgId}/{invoiceId}/claim-{claimId}.pdf`. Signed URL TTL 300s.
   - Create InvoiceInterestClaim row with snapshot values.
   - If `issueAsSecondaryInvoice`: create a new Invoice with `source = 'LATE_INTEREST_CLAIM'`, single InvoiceLine for the claim amount, link via `sourceReference`.
   - Return `{ claimId, pdfUrl, secondaryInvoiceId? }`.

6. `downloadClaim` — `tenantProcedure.input(z.object({ claimId: z.string() })).query(...)`:
   - Look up InvoiceInterestClaim, generate fresh signed URL from R2 pdfKey.
   - Return `{ downloadUrl }`.

**Create `packages/api/src/pdf-templates/late-payment-claim.tsx` (per D-17):**
- Follow the exact pattern of `ir35-sds.tsx` (read it first) — same React-PDF component structure, same design tokens.
- Template content:
  - Header: Organization name, date, "Statutory Late Payment Interest Claim".
  - Body: Original invoice reference, due date, days overdue, principal outstanding, BoE rate used + statutory period, daily accrual, total interest, fixed compensation tier, total claim amount.
  - Footer: locked statutory phrases from `packages/validators/src/legal/gb.ts` — `LPCDA_CLAIM_FOOTER`, `LPCDA_SECTION_REF`.
- All amounts formatted via `minorToDecimalStr` with GBP currency.

**Create `packages/api/src/routers/admin-boe-rate.ts` (per D-10):**
- All procedures gated on `requirePermission('admin:boe-rate:write')` (super-admin only).
1. `list` — returns all BoEBaseRateHistory entries sorted by `effectiveFrom DESC`.
2. `insert` — `z.object({ effectiveFrom: z.date(), ratePercent: z.number().min(0).max(99.99), notes: z.string().optional() })`. Sets `source = 'MANUAL'`, `recordedByUserId` from session. Validates uniqueness on `effectiveFrom`.
3. `update` — `z.object({ id: z.string(), ratePercent: z.number().min(0).max(99.99), notes: z.string().optional() })`. Updates existing row (allows correcting cron-sourced entries).
4. `delete` — `z.object({ id: z.string() })`. Deletes the row.
