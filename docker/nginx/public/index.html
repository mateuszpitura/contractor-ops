Local dev (default ports)

Start stacks with docker compose --profile infisical --profile dev-tooling up -d (adjust ports via env — see docker-compose.yml).

  • Infisical (via nginx gateway)
  • Mailpit (captured email)
  • MinIO Console (S3-compatible API on :9000)
  • CloudBeaver (Postgres — new connection: host infisical-db, port 5432, database infisical, credentials from docker/infisical.env)
  • Redis Insight (add redis://infisical-redis:6379)
